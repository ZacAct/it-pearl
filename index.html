Progressive enhancement is a strategy for web design that emphasizes accessibility, semantic HTML markup, and external CSS and JavaScript. It starts ground up first starting with the content which is the most importnat part of a website. Then it adds styling and interactivity to enhance the user experience. This approach ensures that the website is accessible to all users, regardless of their device or browser, and that it will continue to function even if certain features aren't supported due to an older browser. Website: https://developer.mozilla.org/en-US/docs/Glossary/Progressive_Enhancement
